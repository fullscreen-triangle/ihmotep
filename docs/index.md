### Prerequisites
- Rust 1.70+ with nightly features
- Python 3.9+ with maturin
- CUDA-capable GPU (recommended)
- High-memory system (32GB+ recommended)

### Installation
```bash
git clone https://github.com/fullscreen-triangle/imhotep
cd imhotep
make install
```
